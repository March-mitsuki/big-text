# Big Text
一个在网页上实现**真全屏**滚动显示信息的App。

⭐[https://big-text.mitsuki.app](https://big-text.mitsuki.app)⭐

# 动机
都2025年了我显示个滚动屏我还得安装App啊？这是否有点...

于是就有了这个项目。

# 核心实现
通过渲染一个隐藏在页面中的canvas，并捕捉帧作为video元素的内容来规避网页不能全屏的问题。

因为最终用户全屏的的是video，而video每个浏览器肯定都做了适配一定能全屏。
